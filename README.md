hiyokochan_n11/松風は以下の特徴を備えるマクロパッドです。
- テンキー配列の11キー
- 3つのロータリーエンコーダー（プッシュボタン付き）
- ３つのトグルスイッチによる機能変更（レイヤー変更）
- 現在の機能把握に便利なLEDインジケーター
- 交換可能な和柄プレート（松風のみ）  

ファームウェアはVial対応です。  

vialのソフトウェアから自由にキーマップを変更可能です。
